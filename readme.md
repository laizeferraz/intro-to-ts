# intro-to-ts

An introductory TypeScript workshop for beginners.


## Slides

[bit.ly/muses-code-ts](http://bit.ly/muses-code-ts)

## Setup
download or clone the repo.

### for exercises 1-3:

install dependencies:
```
yarn
```
or
```
npm install
```

### for exercise 4
navigate to app folder and install dependencies
```
cd app
yarn
```
or
```
cd app
npm install
```
